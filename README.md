# About

![With the resource pack enabled](/image/enable.png)

- Vines have different front and back textures

# Require

[![Requires Polytone Mod](/image/require.webp)](https://modrinth.com/mod/polytone)

- This resource pack requires the [Polytone mod](https://modrinth.com/mod/polytone)
- This mod is required for the resource pack to have biome-based colors.
